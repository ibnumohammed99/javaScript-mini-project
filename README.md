# 📝 Interactive To-Do List App

A clean and interactive To-Do List application built using **HTML, CSS, and JavaScript**.  
This project demonstrates fundamental JavaScript concepts such as state management, DOM manipulation, and event-driven programming.

---

## 🚀 Features

- ➕ Add new tasks with validation
- ✔️ Mark tasks as completed/uncompleted
- 🗑️ Delete individual tasks
- 📊 Live task counter (remaining & completed)
- ⚠️ Duplicate task prevention
- 🧹 Clear all tasks option
- 🎨 Dynamic background color picker
- 🎉 Celebration message when all tasks are completed

---

## 🧠 Core Concepts Used

- JavaScript Arrays & Objects (state management)
- DOM Manipulation
- Event Handling
- Functions & re-render system
- Array methods (`push`, `filter`, `splice`, `some`)
- Conditional rendering
